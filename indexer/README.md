# NounSwap Indexer App

NounSwap indexer app built using [ponder](https://ponder.sh/)

## Local Development

Copy `.env.example` to `.env.local`

```bash
cp .env.example .env.local
```

Populate the `.env` file

Install dependencies

```bash
bun i
```

Start dev server for mainnet

```bash
bun dev
```
